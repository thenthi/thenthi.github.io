Reflection

When I first interacted with the Luni website, I realised that the experience felt very different from other traditional websites I had visited before. Instead of following a conventional design that mainly presents text and images, the Luni website uses 3D objects to attract users, and these elements can also be interacted with. Personally, I found myself paying more attention to how the interface responds to my actions. This made the experience feel more playful and less task-oriented, almost like exploring a digital environment rather than simply browsing a website.

Learning Through Building

After interacting with the Luni website, I decided to create my own website. During the development process, I encountered several challenges. At the beginning, I felt quite confused when working with coding languages such as HTML and CSS, as I did not fully understand how they function. This made the process of writing code quite difficult. However, after reviewing the lecture slides and exploring additional learning resources, I gradually developed a clearer understanding. I learned that HTML and CSS are the two foundational technologies used to create and design websites. While they work together, they serve distinct purposes: HTML provides the structure and content, while CSS controls the style and layout.

Overcoming Technical Challenges

Despite gaining a better understanding of HTML and CSS, I still encountered many challenges throughout the development process. I found it difficult to fully understand certain structures and how they functioned, which often led to errors and caused the website to behave in unexpected ways. To overcome these difficulties, I actively sought additional learning resources. I referred to W3Schools to better understand HTML structures and syntax, watched several tutorial videos on YouTube, and also used AI tools to assist me in creating more complex effects. These resources helped me gradually improve my technical skills and become more confident in solving problems during the development process.

Key Takeaway

In conclusion, this experience helped me understand that creating a website is not only about making it visually appealing, but also about understanding how design and code work together. It also showed me the importance of patience and continuous learning when facing technical challenges. Overall, I feel more confident in both my coding skills and my ability to think more critically about design decisions.
